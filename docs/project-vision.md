# Job Assistant

Goal

A personal job-finding assistant that automatically discovers relevant software engineering jobs and sends them to Telegram.

Main features:

- Scheduled job discovery
- Job filtering
- Deduplication
- Telegram notifications
- Save / Ignore / Apply actions
- Application tracking
- Weekly summaries

Requirements:

- Low cost
- Reliable
- Simple to maintain
- No AI required for v1
- No automatic form submission
- Manual application flow
